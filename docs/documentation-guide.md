# Documentation Guide

## Overview

This project uses JSDoc for comprehensive code documentation and automatic documentation generation.

## Documentation Standard

All classes, functions, and methods are documented using JSDoc comments with:
- Description of purpose
- Parameter types and descriptions
- Return value types
- Usage examples
- Event emissions

## Generating Documentation
```bash
npm run docs
```

## JSDoc Tags Used

- `@class` - Class definitions
- `@extends` - Inheritance relationships
- `@description` - Detailed descriptions
- `@constructor` - Constructor documentation
- `@param` - Parameter types and descriptions
- `@returns` - Return value types
- `@type` - Property types
- `@property` - Class properties
- `@example` - Usage examples
- `@private` - Private methods/properties
